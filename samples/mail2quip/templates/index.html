Mail 2 Quip

Do you have emailed alerts you'd like to appear in Quip? IFTTT or Zapier recipes that should go to Quip? Mail2Quip allows you to create new messages by email.

  1. Get a token for the user you want to post as.
  2. Send an email to <thread_id>+<token>@mail2quip.appspotmail.com
  3. See your message posted to that thread.
